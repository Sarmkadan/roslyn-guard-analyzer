# Security Policy

## Supported Versions

The following versions of RoslynGuardAnalyzer are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of RoslynGuardAnalyzer seriously. If you discover a security vulnerability, please let us know right away.

To report a vulnerability, please use **GitHub's built-in private advisory feature**. This is the preferred method for responsible disclosure.

Alternatively, you can email your findings directly to **rutova2@gmail.com**.

### What to Expect

- We will acknowledge your report within **48 hours**.
- We will triage the issue and provide an estimated timeline for a fix.
- We ask that you maintain confidentiality until we have had a chance to address the vulnerability and release an update.
